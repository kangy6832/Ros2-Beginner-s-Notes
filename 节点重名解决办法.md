1.名称重映射
语法： ros2 run 包名 节点名 --ros-args --remap __name:=新名称
或
ros2 run 包名 节点名 --ros-args --remap __node:=新名称

```
ros2 run turtlesim turtlesim_node --ros-args --remp __name:=turtle1
```

2.设置命名空间

```
ros2 run turtlesim turtlesim_node --ros-args --remap __ns:=/t1
```

 语法：ros2 run 包名 节点名 --ros-args --remap __ns:=命名空间

3.设置命名空间与名称重映射叠加
语法: ros2 run 包名 节点名 --ros-args --remap __ns:=新名称 --remap __name:=新名称

```
ros2 run turtlesim turtlesim_node --ros-args --remap __ns:=/t1 --remp __name:=turtle1
```

3.使用launch文件设置
  新建节点（ros2 pkg create cpp_names --build-type ament_cmake --node-name demo01_names），再节点下创建launch文件夹，在CmakeLists.txt文件内install(DIRECTORY launch DESTINATION share/${PROJECT_NAME})
 在launch文件夹下创建文件并编写
    python实现：

```python
from launch import LaunchDescription
         from launch_ros.actions import Node
         def generate_launch_description():
         	return LaunchDescription([
         		Node(package = "turtlesim" , executable = "turtlesim_node" , name = "turtle1"),
         		Node(package = "turtlesim" , executable = "turtlesim_node" , namespace = "t1" ),
         		Node(package = "turtlesim" , executable = "turtlesim_node" , namespace = "t1" , name = "turtle1")
         	])
    
```


```xml
XML实现：
	<launch>
		<node pkg = "turtlesim" exec = "turtlesim_node" name = "turtle1" />
		<node pkg = "turtlesim" exec = "turtlesim_node" namespace = "t1" />
		<node pkg = "turtlesim" exec = "turtlesim_node" name = "turtle1" namespace = "t1" />
	</launch>
	

```

